## ğŸ› ï¸ Cara Menjalankan Game

### Opsi A: Versi Lite (Tanpa Musik & Server)
Jika kamu menggunakan versi kode yang sudah disederhanakan (tanpa audio):
1.  Pastikan file `index.html`, `sketch.js`, dan `p5.min.js` ada di satu folder.
2.  Cukup **Double Click** file `index.html`.
3.  Game akan langsung berjalan di browsermu.

### Opsi B: Versi Full (Dengan Musik. Recommended)
Jika game memiliki fitur Audio (`music.wav`), browser memblokir file suara jika dibuka langsung. Kamu butuh Local Server:
1.  Install **Visual Studio Code (VS Code)**.
2.  Install Ekstensi **"Live Server"** di VS Code.
3.  Buka folder game di VS Code.
4.  Buka `index.html`, Klik Kanan, lalu pilih **"Open with Live Server"**.
